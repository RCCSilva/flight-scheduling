# Template Express

## Scripts

### Build and Deployment

```shell
yarn build # Compiles typescript files to javascript in the "dist" folder
yarn start # Starts node runtime with ./dist/index.js
```

### Development Server

```shell
yarn dev
```

### Create Migrations

```shell
prisma migrate dev --name MigrationName
```

## Techs

- Node
- Express (>=5.0.0)
- Prisma
